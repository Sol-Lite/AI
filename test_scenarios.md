## 3. 이전 대화 맥락 기반 follow-up (agent + history)

> **반드시 순서대로 진행** — 각 시나리오는 1턴 → 2턴 → 3턴 순서로 입력

### 3-A. 포트폴리오 → 종목 비교

| 순서 | 입력 | 기대 동작 | 결과 |
|------|------|-----------|------|
| 1턴 | `포트폴리오 보여줘` | 전체 템플릿 출력 | |
| 2턴 | `제일 많이 오른 종목이 뭐야?` | "SK하이닉스가 +0.49%로 가장 많이 올랐어요." (한 줄) | |
| 3턴 | `그 종목 비중은?` | "SK하이닉스 비중은 78.8%예요." (한 줄) | |
| 4턴 | `현재가는 어때?` | SK하이닉스 현재가 + 등락률 (이전 맥락 종목 유지) | |

### 3-B. 포트폴리오 → 구성 해석

| 순서 | 입력 | 기대 동작 | 결과 |
|------|------|-----------|------|
| 1턴 | `포트폴리오 보여줘` | 전체 템플릿 출력 | |
| 2턴 | `국내랑 해외 비중이 어때?` | 국내/해외 비율 답변 (한 줄) | |
| 3턴 | `아까 본 거 기준으로 리스크가 얼마야?` | MDD, 변동성 수치 답변 | |

### 3-C. 수익률 조회 → 비교

| 순서 | 입력 | 기대 동작 | 결과 |
|------|------|-----------|------|
| 1턴 | `내 수익률 어때` | 기간별 수익률 템플릿 출력 | |
| 2턴 | `1개월이랑 6개월 중 어느 게 더 좋아?` | 두 수치 비교 답변 (한 줄) | |
| 3턴 | `최저 종목은 뭐야?` | worst_stock 종목명 + 수익률 | |

### 3-D. 거래내역 → 통계

| 순서 | 입력 | 기대 동작 | 결과 |
|------|------|-----------|------|
| 1턴 | `거래내역 보여줘` | 최근 거래 목록 출력 | |
| 2턴 | `매수가 더 많아 매도가 더 많아?` | buy_count vs sell_count 비교 | |
| 3턴 | `제일 최근 거래가 뭐야?` | 최근 1건 종목, 금액 답변 | |

### 3-E. 종목 시세 → 연속 조회

| 순서 | 입력 | 기대 동작 | 결과 |
|------|------|-----------|------|
| 1턴 | `삼성전자 시세` | 삼성전자 현재가 출력 | |
| 2턴 | `SK하이닉스는?` | SK하이닉스 시세 조회 (이전 도구 맥락 유지) | |
| 3턴 | `둘 중에 오늘 더 많이 오른 게 어디야?` | 두 종목 등락률 비교 | |

### 3-F. 뉴스 → 연속 조회

| 순서 | 입력 | 기대 동작 | 결과 |
|------|------|-----------|------|
| 1턴 | `삼성전자 뉴스` | 삼성전자 뉴스 출력 | |
| 2턴 | `현대차는?` | 현대차 뉴스 조회 (이전 도구 맥락 유지) | |
