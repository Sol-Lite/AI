## 3. 이전 대화 맥락 기반 follow-up (agent + history)

> **반드시 순서대로 진행** — 각 시나리오는 1턴 → 2턴 → 3턴 순서로 입력

### 3-A. 포트폴리오 → 종목 대명사 follow-up

| 순서 | 입력 | 기대 동작 | 결과 |
|------|------|-----------|------|
| 1턴 | `포트폴리오 보여줘` | 전체 템플릿 출력 | |
| 2턴 | `제일 많이 내린 종목이 뭐야?` | worst_stock 종목명 + 수익률 답변 | |
| 3턴 | `그 종목 현재가` | 2턴에서 답한 종목의 현재가 카드 출력 (stock_price) | |
| 4턴 | `그 종목 비중은?` | 동일 종목 비중 답변 (포트폴리오 맥락 유지) | |
| — | `제일 많이 오른 종목이 뭐야?` | best_stock 종목명 + 수익률 답변 | |
| — | `그 종목 현재가` | 위에서 답한 종목의 현재가 카드 출력 | |

### 3-B. 포트폴리오 → 구성 해석

| 순서 | 입력 | 기대 동작 | 결과 |
|------|------|-----------|------|
| 1턴 | `포트폴리오 보여줘` | 전체 템플릿 출력 | |
| 2턴 | `국내랑 해외 비중이 어때?` | 국내/해외 비율 답변 (한 줄) | |
| 3턴 | `아까 본 거 기준으로 리스크가 얼마야?` | MDD, 변동성 수치 답변 | |

### 3-C. 수익률 조회 → 비교

| 순서 | 입력 | 기대 동작 | 결과 |
|------|------|-----------|------|
| 1턴 | `내 수익률 어때` | 기간별 수익률 템플릿 출력 | |
| 2턴 | `1개월이랑 6개월 중 어느 게 더 좋아?` | 두 수치 비교 답변 (한 줄) | |
| 3턴 | `최저 종목은 뭐야?` | worst_stock 종목명 + 수익률 | |

### 3-D. 거래내역 → 통계

| 순서 | 입력 | 기대 동작 | 결과 |
|------|------|-----------|------|
| 1턴 | `거래내역 보여줘` | 최근 거래 목록 출력 | |
| 2턴 | `매수가 더 많아 매도가 더 많아?` | buy_count vs sell_count 비교 | |
| 3턴 | `제일 최근 거래가 뭐야?` | 최근 1건 종목, 금액 답변 | |

### 3-E. 종목 시세 → 연속 조회

| 순서 | 입력 | 기대 동작 | 결과 |
|------|------|-----------|------|
| 1턴 | `삼성전자 시세` | 삼성전자 현재가 카드 출력 | |
| 2턴 | `SK하이닉스는?` | SK하이닉스 시세 카드 (shortcut: 이전 도구 맥락 유지) | |
| 3턴 | `삼성전자 SK하이닉스 중 더 많이 오른 건?` | "종목명이 하나일 때만 답변이 가능해요." 안내 | |

### 3-F. 뉴스 → 연속 조회 (shortcut + 조사 붙은 입력)

| 순서 | 입력 | 기대 동작 | 결과 |
|------|------|-----------|------|
| 1턴 | `현대차 뉴스` | 현대차 뉴스 카드 출력 | |
| 2턴 | `하닉은` | SK하이닉스 뉴스 카드 (shortcut 발동, 조사 제거 후 해소) | |
| 3턴 | `아마존은` | 아마존닷컴 뉴스 카드 (shortcut 발동) | |

### 3-G. 포트폴리오 맥락 종목 비교 (agent)

> 직전 포트폴리오 tool이 있거나, '보유 종목 중' 문구가 있으면 agent 처리

| 순서 | 입력 | 기대 동작 | 결과 |
|------|------|-----------|------|
| 1턴 | `포트폴리오 보여줘` | 전체 템플릿 출력 | |
| 2턴 | `제일 많이 오른 종목이 뭐야?` | 포트폴리오 조회 후 최고 수익 종목 답변 (agent) | |
| 3턴 | `가장 많이 내린 건?` | 포트폴리오 조회 후 최저 수익 종목 답변 (agent) | |
| — | `보유 종목 중 가장 많이 오른 건?` (세션 무관) | 항상 agent | |

| # | 입력 | 현재 한계 |
|---|------|-----------|
| 6-1 | `포트폴리오에서 삼성전자 비중 얼마야?` | 삼성전자 미보유시 0% 또는 오답 가능 |

| 6-3 | `지난달 거래 중 손실난 게 뭐야?` | 날짜 범위 + 조건 필터 — agent 처리, 불안정 |
| 6-4 | `하닉 시황` (세션 시작 직후) | shortcut 발동 안 됨 → "어떤 정보를 원하시나요?" 안내 |
